Cards
mat-card
Basic card

Material defines some sections that can be used inside this card component :
- card title : mat-card-title
- content : mat-card-content
- user actions : mat-card-actions

Card Title This is the card content. Button can be aligned to the right with align="end" on 'mat-card-actions' component.

There are other sections that allows to build more complex card components. See documentation.
