Date Picker
mat-form-field > input + mat-datepicker-toggle + mat-datepicker

Material import :
MatDatepickerModule
MatNativeDateModule

Basically, a datepicker is composed of a text input and a calendar pop-up
- Add a datepicker toggle, with matSuffix attribute (suffix icon in the input)
- Add the datepicker component

Then, link the 3 elements together :
- mat-datepicker : create a ref variable #myDatePicker
- input : [matDatepicker]="myDatePicker"
- mat-datepicker-toggle : [for]="myDatePicker"



Configurations

Starting view of the calendar : by default Month View
datepicker *startView : year (months in year), multi-year (multiple years)



Minimum and maximum date
Use two date objects in component.ts : min and max
Bind to the input component : [min]="minDate" [max]="maxDate"





Filter dates : i.e. sundays and saturdays
- component.ts : create a dateFilter function
- input element : [matDatepickerFilter]="dateFilter"
