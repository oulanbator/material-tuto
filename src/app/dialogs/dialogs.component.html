Dialogs
See dialog-example component

A component used as a dialog (dialog-example) has to be included in the entryComponents array (app.module.ts)

There are several directive to structure the dialog component, i.e :
mat-dialog-close : used on a button for closing the dialog

Catch the dialog reference in a variable (dialogRef), and use it to subscribe to the observable
I.e : perform actions on closing the dialog
Finally we assign a value to the mat-dialog-close directive, which is accessed as the result.
We can then use this value in the component.ts regarding the value returned (true or false i.e.)

Last point to discuss is passing data to the dialog.
First step : pass the data as a 2nd parameter to the dialog open method
Second step : use the data is the dialog component (dialog-example.component.ts) :
To use the data in the dialog, we have to use the MAT_DIALOG_DATA injection component
Import @Inject, Import MAT_DIALOG_DATA, add it to constructor(@Inject(MAT_DIALOG_DATA) public data: any)
So basically we are now making the Dialog component capable of receiving any type of data.
Now we can simply interpolate the data object in the HTML of the component.

If you want to specify width and height for the dialog, we can pass in the configuration object.
There are many other properties we can use, based on requirements : check documentation for more info.
