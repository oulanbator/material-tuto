Menus
'mat-menu' component > button (with 'mat-menu-item' attribute)

By itself, the 'mat-menu' component doesn't render anything
It has to be triggered by UI element or programmatically

Let's add a button to trigger the menu. We associate :
- use a template reference var (appMenu) with a directive in the 'mat-menu' component
- link the button with the template reference var (appMenu) with matMenuTriggerFor directive

Lazy rendering

By default the menu choices are initialized before the menu is open.
We can differ this init, until menu is opened, using ngTemplate directive and matMenuContent attribute. Allows us to have dynamic content as menus choices.
