Hide/Show sidenav

Sidenav always hidden by default : 3 methods

Method 1
- Can use the "opened" attribute on mat-sidenav component
- Better is to bind this attribute to a property so we can trigger and toggle the sidenav

Method 2
Another way is to call open() and close() methods on template reference variable of the #sidenav

Method 3
Toggle the sidenav using the sidenav.toggle() method

Hide/Show events

Finally if we want to perform actions upon the opening, and closing of he sidenav, we can listen for the '(opened)' and '(closed)' events. Access directly in the mat-sidenav component, and pass a callback function : here we call the log() method.

Different modes for sidenav

Modes : specified with the "mode" attribute on mat-sidenav component (markup tag)
- mode="over" (default)
- mode="push" (push the content to the right)
- mode="side" (push to the right, but content and sidenav are both usable side by side)
