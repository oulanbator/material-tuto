Create a button to call the snackbar

We need to create the method in component.ts to show the snackBar, using the service MatSnackBar injected in the constructor
We also can specify an action for the snackBar. We pass it as 2nd argument to the method.
By default, clicking the action, will close the snackbar.
Sometimes, you want this action to be "Undo" of an action

We are alerted that the snackBar is dismissed, or undo, is through observables
we create a reference to the opened snackbar (let snackBarRef = )
So, using this ref, we can subscribe to "afterDismissed" or "onAction" observables.
Then, we can put the code we want...

There is a 3rd parameter to the snackbar service (component.ts), that accepts some configuration in an object
The most used : 'duration' property (time before auto-dismiss of the snackbar)
