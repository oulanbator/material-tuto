We can convert the stepper to a linear stepper to avoid the user going through step without restrictions
1 - add the linear attribute to mat-horizontal-stepper component
2 - add a completed attribute to each individual step (default to false)
Here it is hard coded, but ideally we should use property binding in order to control fulfilment of each step.
If interaction is completed, call a method that update completed property to true and allow the user to go further.
You can also use the attribute optional on a step
